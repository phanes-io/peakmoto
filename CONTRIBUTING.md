# Contributing to PeakMoto

Thanks for your interest in contributing! PeakMoto is a community-driven project.

## Getting Started

1. Fork the repo
2. Clone your fork
3. Run `flutter pub get`
4. Create a feature branch
5. Make your changes
6. Run `flutter test`
7. Open a PR

## Guidelines

- Follow existing code style
- Keep PRs focused on a single change
- Write tests for new features
- Use German or English for commit messages
